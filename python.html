Python

Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. Its main uses are web development, data science, artificial intelligence, and automation. Here you will learn the basics of Python programming.

Comments

Comments in Python are lines that start with a # symbol and are ignored by the interpreter. They are used to explain code and make it more readable.

# This is a comment in Python

Variables

Types of variables

Type Use Example
Integer Storing whole numbers x = 5
Float Storing decimal numbers y = 3.14
String Storing text name = "John"
Boolean Storing True or False values is_active = True

Strings

Strings are used to store text. They can be any length or single characters. They are enclosed in either single or double quotes. They can be created by assigning a value to a variable.

# Creating a string
my_string = "Hello, World!"
print(my_string)

# Output: Hello, World!

There are a lot of useful operations that can be performed on strings in Python.

# Concatenation
str1 = "Hello, "
str2 = "World!"
result = str1 + str2
print(result)  # Output: Hello, World!

# f strings
name = "Alice"
age = 30
message = f"My name is {name} and I am {age} years old."
print(message)  # Output: My name is Alice and I am 30 years old.

# Length
length = len(str1)
print(length)  # Output: 7

# Indexing
first_char = str1[0]
print(first_char)  # Output: H

# Slicing
substring = str2[0:5]
print(substring)  # Output: World

# Splitting
words = str1.split(", ")
print(words)  # Output: ['Hello', 'World!']

# Repetition
str3 = "Ha" * 3
print(str3)  # Output: HaHaHa

Integers and Floats

Integers are whole numbers, while floats are decimal numbers. They are both numeric data types in Python.

# Creating an integer
my_integer = 10

# Creating a float
my_float = 3.14

# Addition
sum = my_integer + my_float
print(sum)  # Output: 13.14

# Subtraction
difference = my_integer - my_float
print(difference)  # Output: 6.86

# Multiplication
product = my_integer * my_float
print(product)  # Output: 31.4

# Division
quotient = my_integer / my_float
print(quotient)  # Output: 0.92

Booleans

Booleans are a data type that can only have two values: True or False. They are often used in conditional statements and logical operations.

# Creating a boolean
is_active = True
is_admin = False

# Logical AND
result_and = is_active and is_admin
print(result_and)  # Output: False

# Logical OR
result_or = is_active or is_admin
print(result_or)  # Output: True

# Logical NOT
result_not = not is_active
print(result_not)  # Output: False

Data Types

Python has several built-in data types, including:

  • Integer (int): Whole numbers, e.g., 1, 2, 3
  • Float (float): Decimal numbers, e.g., 3.14, 2.71
  • String (str): Text enclosed in quotes, e.g., "Hello"
  • Boolean (bool): True or False values
  • List: Ordered collection of items, e.g., [1, 2, 3]
  • Tuple: Immutable ordered collection of items, e.g., (1, 2, 3)
  • Dictionary (dict): Key-value pairs, e.g., {"name": "John", "age": 30}
  • Set: Unordered collection of unique items, e.g., {1, 2, 3}

This chapter will only cover Lists, Dictionarys, Sets and Tuples since the others have been covered in the previous chapter.


Lists

Lists are ordered collections of items that can be of different data types. They are mutable, meaning you can change their content.

# Creating a list
my_list = [1, 2, 3, "Hello", True]

# Accessing single elements
first_element = my_list[0]
print(first_element)  # Output: 1

# Accessing a range of elements
subset = my_list[1:4]
print(subset)  # Output: [2, 3, "Hello"]

# Modifying elements
my_list[2] = 5
print(my_list)  # Output: [1, 2, 5, "Hello", True]

Dictionaries

Dictionaries are unordered collections of key-value pairs. Keys must be unique and immutable, while values can be of any data type.

# Creating a dictionary
my_dict = {"name": "John", "age": 30, "city": "New York"}

# Accessing values
name = my_dict["name"]
print(name)  # Output: John

# Modifying values
my_dict["age"] = 31
print(my_dict)  # Output: {"name": "John", "age": 31, "city": "New York"}

Sets

Sets are unordered collections of unique elements. They are useful for removing duplicates and performing mathematical set operations.

# Creating a set
my_set = {1, 2, 3, 4, 5}

# Adding elements
my_set.add(6)
print(my_set)  # Output: {1, 2, 3, 4, 5, 6}

# Removing elements
my_set.remove(2)
print(my_set)  # Output: {1, 3, 4, 5, 6}

Tuples

Tuples are ordered collections of items that are immutable (cannot be changed). They are similar to lists but cannot be modified after creation.

# Creating a tuple
my_tuple = (1, 2, 3, "Hello", True)

# Accessing elements
first_element = my_tuple[0]
print(first_element)  # Output: 1

# Tuples are immutable - this would cause an error:
# my_tuple[0] = 5

Operators

Operators are special symbols that perform operations on variables and values. Python has several types of operators, including:

  • Arithmetic Operators: +, -, *, /, %, **, //
  • Comparison Operators: ==, !=, >, <,>=, <=< /li>
  • Logical Operators: and, or, not
  • Assignment Operators: =, +=, -=, *=, /=
  • Membership Operators: in, not in
  • Identity Operators: is, is not

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations on numbers.

# Addition
result_add = 5 + 3
print(result_add)  # Output: 8

# Subtraction
result_sub = 5 - 3
print(result_sub)  # Output: 2

# Multiplication
result_mul = 5 * 3
print(result_mul)  # Output: 15

# Division
result_div = 5 / 3
print(result_div)  # Output: 1.6666666666666667

# Modulus (remainder)
result_mod = 5 % 3
print(result_mod)  # Output: 2

# Exponentiation
result_exp = 5 ** 3
print(result_exp)  # Output: 125

# Floor Division (quotient rounded down to the nearest integer)
result_floor = 5 // 3
print(result_floor)  # Output: 1

Comparison Operators

Comparison operators are used to compare values and return a boolean result (True or False). 

# Equal to
result_eq = 5 == 3
print(result_eq)  # Output: False

# Not equal to
result_ne = 5 != 3
print(result_ne)  # Output: True

# Greater than
result_gt = 5 > 3
print(result_gt)  # Output: True

# Less than
result_lt = 5 < 3
print(result_lt)  # Output: False

# Greater than or equal to
result_ge = 5 >= 3
print(result_ge)  # Output: True

# Less than or equal to
result_le = 5 <= 3
print(result_le)  # Output: False

Logical Operators

Logical operators are used to combine conditional statements and return a boolean result. 

# Logical AND
result_and = True and False
print(result_and)  # Output: False

# Logical OR
result_or = True or False
print(result_or)  # Output: True

# Logical NOT
result_not = not True
print(result_not)  # Output: False

Assignment Operators

Assignment operators are used to assign values to variables.

# Simple assignment
x = 5
print(x)  # Output: 5

# Addition assignment
x += 3
print(x)  # Output: 8

# Subtraction assignment
x -= 3
print(x)  # Output: 5

# Multiplication assignment
x *= 3
print(x)  # Output: 15

# Division assignment
x /= 3
print(x)  # Output: 5.0

Membership Operators

Membership operators are used to check if a value is present in a sequence (like a list, tuple, or string).

# in operator
my_list = [1, 2, 3, 4, 5]
result_in = 3 in my_list
print(result_in)  # Output: True

# not in operator
result_not_in = 6 not in my_list
print(result_not_in)  # Output: True

Identity Operators

Identity operators are used to check if two variables refer to the same object in memory. 

# is operator
x = [1, 2, 3]
y = x
result_is = x is y
print(result_is)  # Output: True

# is not operator
result_is_not = x is not y
print(result_is_not)  # Output: False

Conditional Statements

Conditional statements are used to perform different actions based on different conditions. The most common conditional statements in Python are if, elif, and else.


if statement

if statements are used to execute a block of code if a specified condition is true.

# if statement
x = 10
if x > 5:
    print("x is greater than 5")

elif statement

elif statements are used to check another condition if the previous if statement is false. 

# elif statement
x = 10
if x > 15:
    print("x is greater than 15")
elif x > 5:
    print("x is greater than 5 but less than or equal to 15")

else statement

else statements are used to execute a block of code if all previous conditions are false. 

# else statement
x = 10
if x > 15:
    print("x is greater than 15")
elif x > 5:
    print("x is greater than 5 but less than or equal to 15")
else:
    print("x is less than or equal to 5")

Match Statements

Match statements are used to compare a value against a series of patterns and execute the corresponding block of code when a match is found. They are similar to switch statements in other programming languages.

# Match statement
x = 3
match x:
    case 1:
        print("x is 1")
    case 2:
        print("x is 2")
    case 3:
        print("x is 3")
    case _:
        print("x is not 1, 2, or 3")

Loops

Loops are used to execute a block of code repeatedly until a certain condition is met. The most common loops in Python are for and while loops.


for loop

for loops are used to iterate over a sequence (like a list, tuple, or string) and execute a block of code for each item in the sequence.

# for loop
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Output:
# apple
# banana
# cherry

while loop

while loops are used to execute a block of code repeatedly as long as a specified condition is true.

# while loop
count = 0
while count < 5:
    print(count)
    count += 1

# Output:
# 0
# 1
# 2
# 3
# 4

Functions

Functions are reusable blocks of code that perform a specific task. They can take parameters and return values.

# Function definition
def greet(name):
    print(f"Hello, {name}!")

# Function call
greet("Alice")

# Output:
# Hello, Alice!

Try - Except

Try - Except blocks are used to handle exceptions (errors) that may occur during the execution of a program. The code inside the try block is executed, and if an exception occurs, the code inside the except block is executed.

# Try - Except block
try:
    result = 10 / 0  # This will raise a ZeroDivisionError
except ZeroDivisionError:
    print("You cannot divide by zero!")

# Output:
# You cannot divide by zero!

Classes

Classes are blueprints for creating objects. They define a set of attributes and methods that the objects created from the class will have.

# Class definition
class Car:
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model

    def drive(self):
        print(f"The {self.brand} {self.model} is driving.")

# Creating an object from the class
my_car = Car("Toyota", "Corolla")
        my_car.drive()  # Output: The Toyota Corolla is driving.
