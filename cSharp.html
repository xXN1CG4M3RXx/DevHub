C#

Introduction

C# is a modern, object-oriented programming language developed by Microsoft. It is widely used for building applications on the .NET platform, including web applications, desktop applications, and mobile apps.

Comments

Comments are used to explain code and make it more readable. In C#, you can use single-line comments with // or multi-line comments with /* */.

// This is a single-line comment 

/* This is a multi-line comment
that spans multiple lines */

Strings

Strings are used to store text. In C#, you can create a string variable using the string keyword.

// Creating a string
string greeting = "Hello, World!";
Console.WriteLine(greeting);

// Output: Hello, World!

There are a lot of string methods available in C# that allow you to manipulate and work with strings. Some of the most commonly used methods include:

  • Length: Returns the length of the string.
  • Substring: Extracts a portion of the string.
  • Replace: Replaces occurrences of a specified substring with another substring.
  • ToUpper: Converts the string to uppercase.
  • ToLower: Converts the string to lowercase.
  • Trim: Removes leading and trailing whitespace from the string.

string name = "John";
int age = 30;
string message = $"My name is {name} and I am {age} years old.";
Console.WriteLine(message); // Output: My name is John and I am 30 years old.

// Concatenating strings
string firstName = "John";
string lastName = "Doe";
string fullName = firstName + " " + lastName;
Console.WriteLine(fullName); // Output: John Doe

// Length of a string
string text = "Hello, World!";
int length = text.Length;
Console.WriteLine(length); // Output: 13

// Substring
string substring = text.Substring(7, 5);
Console.WriteLine(substring); // Output: World

// Replace
string replaced = text.Replace("World", "Universe");
Console.WriteLine(replaced); // Output: Hello, Universe!

// ToUpper and ToLower
string upper = text.ToUpper();
string lower = text.ToLower();
Console.WriteLine(upper); // Output: HELLO, WORLD!
Console.WriteLine(lower); // Output: hello, world!

// Trim
string trimmed = "   Hello, World!   ".Trim();
Console.WriteLine(trimmed); // Output: Hello, World!

Integers and Doubles

Integers and doubles are used to store whole numbers and decimal numbers, respectively.

// Creating an integer
int myInt = 42;

// Creating a double
double myDouble = 3.14;

// Addition
int sum = myInt + 10;
Console.WriteLine(sum); // Output: 52

// Subtraction
int difference = myInt - 10;
Console.WriteLine(difference); // Output: 32

// Multiplication
int product = myInt * 2;
Console.WriteLine(product); // Output: 84

// Division
double quotient = myDouble / 2;
Console.WriteLine(quotient); // Output: 1.5

Booleans

Booleans are used to store True or False values. In C#, you can create a boolean variable using the bool keyword.

// Creating a boolean
bool isActive = true;
bool isInactive = false;

// Logical AND
bool andResult = isActive && isInactive;
Console.WriteLine(andResult); // Output: false

// Logical OR
bool orResult = isActive || isInactive;
Console.WriteLine(orResult); // Output: true

// Logical NOT
bool notResult = !isActive;
Console.WriteLine(notResult); // Output: false

Data Types

In C#, data types define the type of value a variable can hold. Common data types include:

  • int: Whole numbers
  • double: Decimal numbers
  • string: Text
  • bool: True or False
  • array: A collection of elements of the same type
  • var: Implicitly typed variable (the type is inferred by the compiler)

Arrays

Arrays are used to store multiple values of the same type. In C#, you can create an array using the following syntax:

// Creating an array of integers
int[] numbers = new int[5];

// Initializing an array with values
string[] names = new string[] { "Alice", "Bob", "Charlie" };

// Modifying an array element
numbers[0] = 10;
numbers[2] = 30;
numbers[4] = 50;

// Accessing array elements
Console.WriteLine(names[0]); // Output: Alice

Operators

Operators are symbols that perform operations on variables and values. Common operators in C# include:

  • Arithmetic Operators: +, -, *, /, %
  • Comparison Operators: ==, !=, >, <,>=, <=< /li>
  • Logical Operators: &&, ||, !
  • Assignment Operators: =, +=, -=, *=, /=, %=

Conditional Statements

Conditional statements allow you to execute different blocks of code based on certain conditions. In C#, you can use if, else if, and else statements for conditional logic. There are 4 types of conditional statements:

  • If Statement: Executes a block of code if a specified condition is true.
  • Else If Statement: Specifies a new condition to test if the previous condition is false.
  • Else Statement: Executes a block of code if all previous conditions are false.
  • Switch Statement: Selects one of many code blocks to execute based on the value of a variable.

If Statement

Executes a block of code if a specified condition is true

int number = 10;
if (number > 0) {
    Console.WriteLine("The number is positive.");
}

Else If Statement

Executes a block of code if the previous condition is false and a new condition is true

int number = -15;
if (number > 0) {
    Console.WriteLine("The number is positive.");
} else if (number < 0) {
    Console.WriteLine("The number is negative.");
}

Else Statement

Executes a block of code if all previous conditions are false

int number = 0;
if (number > 0) {
    Console.WriteLine("The number is positive.");
} else if (number < 0) {
    Console.WriteLine("The number is negative.");
} else {
    Console.WriteLine("The number is zero.");
}

Switch Statement

Selects one of many code blocks to execute based on the value of a variable

int day = 3;
switch (day) {
    case 1:
        Console.WriteLine("Monday");
        break;
    case 2:
        Console.WriteLine("Tuesday");
        break;
    case 3:
        Console.WriteLine("Wednesday");
        break;
    default:
        Console.WriteLine("Invalid day");
        break;
}

Loops

Loops are used to execute a block of code repeatedly until a specific condition is met. In C#, there are four types of loops:

  • For Loop: Executes a block of code a specified number of times.
  • While Loop: Executes a block of code as long as a specified condition is true.
  • Do-While Loop: Executes a block of code at least once, and then continues to execute as long as a specified condition is true.
  • Foreach Loop: Iterates through each element in a collection or array.

For Loop

Executes a block of code a specified number of times

for (int i = 0; i < 5; i++) {
    Console.WriteLine(i);
}

While Loop

Executes a block of code as long as a specified condition is true

int i = 0;
while (i < 5) {
    Console.WriteLine(i);
    i++;
}

Do-While Loop

Executes a block of code at least once, and then continues to execute as long as a specified condition is true

int i = 0;
do {
    Console.WriteLine(i);
    i++;
} while (i < 5);

Foreach Loop

Iterates through each element in a collection or array

string[] names = { "Alice", "Bob", "Charlie" };
foreach (string name in names) {
    Console.WriteLine(name);
}

Functions

Functions are reusable blocks of code that perform a specific task. In C#, you can define a function using the following syntax:

return_type FunctionName(parameters) {
    // Function body
}

Example

void Greet(string name) {
    Console.WriteLine($"Hello, {name}!");
}

Greet("Alice"); // Output: Hello, Alice!

Try - Catch

Try-catch blocks are used to handle exceptions and errors in C#. The try block contains code that may throw an exception, while the catch block contains code to handle the exception if it occurs.

try {
    // Code that may throw an exception
}
catch (Exception ex) {
    // Code to handle the exception
}

Example

try {
    int[] numbers = { 1, 2, 3 };
    Console.WriteLine(numbers[5]); // This will throw an IndexOutOfRangeException
}
catch (IndexOutOfRangeException e) {
    Console.WriteLine("An error occurred: " + e.Message);
}

Classes

Classes are blueprints for creating objects in C#. They encapsulate data and behavior related to that data. You can define a class using the following syntax:

class ClassName {
    // Class members and methods
}

Example

class Person {
    public string Name { get; set; }
    public int Age { get; set; }

    public void Greet() {
        Console.WriteLine($"Hello, my name is {Name} and I am {Age} years old.");
    }
}
