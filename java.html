Java

Introduction

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended for writing portable applications that can run on any platform that supports Java.


Comments

Comments are used to explain code and make it more readable. In Java, there are three types of comments:

  • Single-line comments start with // and are used to comment out a single line of code.
  • Multi-line comments start with /* and end with */ and are used to comment out multiple lines of code.
  • Documentation comments start with /** and end with */ and are used to generate documentation for the code.

// This is a single-line comment

/* This is a multi-line comment
that spans multiple lines */

/**
 * This is a documentation comment
 * that explains the purpose of the code
 */

Strings

Strings are used to store text. They can be any length or single characters. They are enclosed in either single or double quotes. They can be created by assigning a value to a variable.

// Creating a string
String myString = "Hello, World!";
System.out.println(myString);

// Output: Hello, World!

There are a lot of useful operations that can be performed on strings in Java.

// Concatenation
String firstName = "John";
String lastName = "Doe";
String fullName = firstName + " " + lastName;
System.out.println(fullName); // Output: John Doe

// Length
String text = "Hello, World!";
int length = text.length();
System.out.println(length); // Output: 137

// Indexing
String str = "Hello, World!";
char firstChar = str.charAt(0);
System.out.println(firstChar); // Output: H

// Substring
String str = "Hello, World!";
String substring = str.substring(7);
System.out.println(substring); // Output: World!

// Replace
String original = "Hello, World!";
String newString = original.replace("World", "Java");
System.out.println(newString); // Output: Hello, Java!

// Repetition
String str = "Hello";
String repeated = str.repeat(3);
System.out.println(repeated); // Output: HelloHelloHello

Integers and Doubles

Integers and doubles are used to store numbers. Integers are whole numbers, while doubles are decimal numbers. They can be created by assigning a value to a variable.

// Creating an integer
int myInt = 42;

// Creating a double
double myDouble = 3.14;

System.out.println(myInt); // Output: 42
System.out.println(myDouble); // Output: 3.14

// Addition
int sum = myInt + 10;
System.out.println(sum); // Output: 52

// Subtraction
int difference = myInt - 10;
System.out.println(difference); // Output: 32

// Multiplication
int product = myInt * 2;
System.out.println(product); // Output: 84

// Division
double quotient = myDouble / 2;
System.out.println(quotient); // Output: 1.57

Booleans

Booleans are used to store True or False values. They can be created by assigning a value to a variable.

// Creating a boolean
boolean isJavaFun = true;
boolean isPythonFun = false;

System.out.println(isJavaFun); // Output: true
System.out.println(isPythonFun); // Output: false

// Logical AND
boolean andResult = isJavaFun && isPythonFun;
System.out.println(andResult); // Output: false

// Logical OR
boolean orResult = isJavaFun || isPythonFun;
System.out.println(orResult); // Output: true

// Logical NOT
boolean notResult = !isJavaFun;
System.out.println(notResult); // Output: false

Data Types

Java has several built-in data types, including:

  • byte: A 8-bit signed integer.
  • short: A 16-bit signed integer.
  • int: A 32-bit signed integer.
  • long: A 64-bit signed integer.
  • float: A 32-bit floating-point number.
  • double: A 64-bit floating-point number.
  • char: A single 16-bit Unicode character.
  • boolean: A value that can be either true or false.
  • String: A sequence of characters.
  • Array: A collection of elements of the same type.

There are also other data types you can use by importing additional libraries or creating custom classes.


Arrays

Arrays are used to store multiple values in a single variable. They can be created by specifying the type of elements and the size of the array.

// Creating an array of integers
int[] myArray = new int[5];

// Assigning values to the array
myArray[0] = 10;
myArray[1] = 20;
myArray[2] = 30;
myArray[3] = 40;
myArray[4] = 50;

// Accessing values from the array
System.out.println(myArray[0]); // Output: 10
System.out.println(myArray[2]); // Output: 30

// Creating an array of strings
String[] names = {"Alice", "Bob", "Charlie"};

// Accessing values from the array
System.out.println(names[1]); // Output: Bob

Operators

Operators are used to perform operations on variables and values. Java has several types of operators, including:

  • Arithmetic operators: Used to perform mathematical operations.
  • Assignment operators: Used to assign values to variables.
  • Comparison operators: Used to compare two values.
  • Logical operators: Used to combine conditional statements.
  • Bitwise operators: Used to perform bit-level operations.
  • Unary operators: Used to perform operations on a single operand.

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations. They include:

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus (remainder)

Assignment Operators

Assignment operators are used to assign values to variables. They include:

  • =: Assigns a value to a variable
  • +=: Adds a value to a variable and assigns the result to that variable
  • -=: Subtracts a value from a variable and assigns the result to that variable
  • *=: Multiplies a variable by a value and assigns the result to that variable
  • /=: Divides a variable by a value and assigns the result to that variable
  • %=: Finds the remainder of a variable divided by a value and assigns the result to that variable

Comparison Operators

Comparison operators are used to compare two values. They include:

  • ==: Checks if two values are equal
  • !=: Checks if two values are not equal
  • >: Checks if the left value is greater than the right value
  • <: Checks if the left value is less than the right value
  • >=: Checks if the left value is greater than or equal to the right value
  • <=: Checks if the left value is less than or equal to the right value

Logical Operators

Logical operators are used to combine conditional statements. They include:

  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

Bitwise Operators

Bitwise operators are used to perform bit-level operations. They include:

  • &: Bitwise AND
  • |: Bitwise OR
  • ^: Bitwise XOR
  • ~: Bitwise NOT
  • <<: Left shift
  • >>: Right shift

Unary Operators

Unary operators are used to perform operations on a single operand. They include:

  • +: Unary plus
  • -: Unary minus
  • !: Logical NOT

Conditional Statements

Conditional statements are used to perform different actions based on different conditions. They include:

  • if
  • else
  • else if
  • switch

if

Executes a block of code if a condition is true.

int x = 20;
if (x > 18) {
    System.out.println("x is greater than 18");
}

else

Executes a block of code if the condition in the if statement is false.

int time = 20;
if (time < 18) {
    System.out.println("Good day.");
} else {
    System.out.println("Good evening.");
}

else if

Executes a block of code if the first condition is false and the else if condition is true. 

int time = 22;
if (time < 10) {
    System.out.println("Good morning.");
} else if (time < 20) {
    System.out.println("Good day.");
} else {
    System.out.println("Good evening.");
}

switch

Executes a block of code based on the value of a variable.

int day = 3;
switch (day) {
    case 1 -> System.out.println("Monday");
    case 2 -> System.out.println("Tuesday");
    case 3 -> System.out.println("Wednesday");
    default -> System.out.println("Another day");
}

Loops

Loops are used to repeatedly execute a block of code until a specific condition is met. They include:

  • for
  • while
  • do-while

For Loop

The for loop is used to iterate over a block of code a specific number of times.

for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

While Loop

The while loop is used to repeatedly execute a block of code as long as a specific condition is true

int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

Functions

In Java, functions are called methods and must be declared inside a class.

public class Main {
    static void myMethod(String name) {
        System.out.println("Hello " + name);
    }

    public static void main(String[] args) {
        myMethod("Alice");
    }
}

Try - Catch

Try-catch blocks are used to handle exceptions and prevent the program from crashing.

try {
    int[] myNumbers = {1, 2, 3};
    System.out.println(myNumbers[10]);
} catch (Exception e) {
    System.out.println("Something went wrong.");
} finally {
    System.out.println("The 'try catch' is finished.");
}

Classes

Java is an object-oriented language. Everything is associated with classes and objects.

public class Car {
    String brand;
    int year;

    // Constructor
    public Car(String brand, int year) {
        this.brand = brand;
        this.year = year;
    }

    public void drive() {
        System.out.println("The " + brand + " is driving!");
    }
}

// Usage
Car myCar = new Car("Tesla", 2023);
myCar.drive();
